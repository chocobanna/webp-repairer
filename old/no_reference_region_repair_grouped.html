Cut Settings

Default is 5 columns × 5 rows. Use the real number of shuffled pieces if you know it. Wrong grid = garbage output, no algorithm can psychic its way out of that.

No-Reference Solver

The draft solver builds its own fake reference by clustering strong matches first, anchoring detailed pieces, then filling weak/blank pieces last. You can edit the result manually.

Edit Mode

Swap: click two output tiles.
Assign: click destination on output, then source tile on the right.

Map

Use locks after you manually fix a region. Refinement will not move locked output slots.

Groups

Groups keep a region together as one movable block. Unlike locks, a group can still relocate during refinement. Use this when a cluster is internally correct but still sitting in the wrong place.
